== README

This is a template page app for prelaunch of any start-up to get started with user traction.

This app has the following features:

* Facebook-login integration

* Data-entry form (for the users who do not wish to be connected with facebook)

* Flexibility to configure the page and change the design usin bootstrap framework

* User database (which can be used by sales or marketing team to grow the business)


Technologies used:

 * Ruby on Rails, Bootstrap
